# 🛍️ Customer Shopping Analysis Dashboard

## 📊 Project Description
This project is an interactive dashboard developed using Power BI to analyze customer shopping behavior and overall sales performance. It provides insights into revenue, total orders, and product categories.

The dashboard helps in understanding customer preferences, payment methods, and shopping trends across different locations. It also includes analysis based on gender and age group.

Monthly revenue trends are visualized to track business performance and compare with previous months. The project uses interactive filters and slicers for better user experience.

Overall, this project demonstrates data visualization and business intelligence skills by converting raw data into meaningful insights.

## 🛠️ Tools Used
- Power BI Desktop  
- DAX (Data Analysis Expressions)  
- Data Modeling  
- Data Visualization  

## 📌 Features
- Category-wise sales analysis  
- Customer demographic insights  
- Payment method analysis  
- Monthly revenue trends  
- Interactive dashboard
